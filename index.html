Create git branch, merge branches [advanced]

go to Terminal and >lower arrow and click bash and command

    Command----------Description
  1. git branch => List branches
  2. git branch [branch name] => Create a new branch
  3. git branch -d [branch name]=> Delete a branch
  4. git checkout [branch name] => Switch to a branch
  5. git checkout -b [branch name] => Create a new branch and switch to it
  6. git merge [branch name] => Merge a branch into the active branch
  7. git merge [source branch] [target branch] => Merge a branch into a target branch
